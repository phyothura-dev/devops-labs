# DevOps Practice Lab

A collection of DevOps experiments using different tech stacks.

## 🚀 Projects

- **Flask**: Dockerized Python app with MySQL and GitHub Actions (CI).
- **Laravel**: PHP application.
- **Odoo**: ERP application.

## 🛠️ Usage

To run the Flask lab:

```bash
cd flask
docker-compose up -d
```
